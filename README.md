Microtronix PCB Quote Workflow
Internal quoting system for Microtronix Manufacturing.
Live app → https://microtronix-pcb.github.io/quotes/
---
Access
Open the link above in any browser. Add to home screen for app-like access:
Android/Chrome — tap "Install" banner at the bottom
iPhone/iPad — tap Share → Add to Home Screen
PC — look for the install icon in the browser address bar
Deployment
This is a single-file HTML app hosted on GitHub Pages. To update:
Replace `microtronix_quote_workflow.html` with the new version
Rename it to `index.html` in the repo root
Commit and push — GitHub Pages auto-deploys in ~1 minute
